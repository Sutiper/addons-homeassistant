# Portainer Business Edition

Interface de gestion Docker complète avec support multi-environnement et Edge Agent.

## Accès

- **HTTP** : `http://<ip-home-assistant>:9000`
- **HTTPS** : `https://<ip-home-assistant>:9443`

## Premier démarrage

Portainer demande la création d'un compte administrateur dans les **5 premières minutes**.
Passé ce délai, redémarre l'addon.

## Edge Agent

Le port **8000** permet la connexion d'agents Edge distants.

## Données

La configuration est persistante dans `/data` entre les redémarrages.
